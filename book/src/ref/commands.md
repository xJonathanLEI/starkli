# Commands

Starkli offers the following commands:

- selector
- class-hash
- to-cairo-string
- parse-cairo-string
- mont
- call
- transaction
- block-number
- block-hash
- block
- block-time
- state-update
- transaction-receipt
- chain-id
- balance
- nonce
- storage
- class-hash-at
- class-by-hash
- class-at
- syncing
- signer
- account
- invoke
- declare
- deploy
- upgrade
- completions

To check usage of each command, run with the `--help` option.
